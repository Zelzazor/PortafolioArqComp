¿Qué tipos de arquitectura del computador existen?

Puedo destacar dos: La arquitectura Von Neumann y la Arquitectura Harvard.

¿Qué es un opcode?

También llamado código de operación, es el código que guarda una instrucción del CPU.

¿Qué son las interrupciones 21?

Es una interrupción que tiene como objetivo llamar a determinadas funciones de DOS. Si por ejemplo se llama a la interrupción 21 conocida como 4C, este se interpreta como la finalización del programa. Es decir, ejecutar esta interrupción finaliza el programa.

¿Qué es un emulador?

Un emulador es un programa que tiene por objetivo mimetizar el comportamiento de otro sistema, el cual posee instrucciones diferentes, con el objetivo de crear un subsistema que permita ejecutar programas compatibles con el sistema original mimetizado.

¿Qué son registros?

Los registros son espacios de memoria dentro del procesador que se emplean para manipular los datos provenientes de la memoria principal de la forma más eficiente.
